# PayDiverse Analytics Dashboards

## Contents
This folder contains executive dashboards for PayDiverse 2.0.

Dashboards are built in Salesforce UI using specifications from:
```
PayDiverse_Dashboards_Spec.md
```

## Dashboards:
1. **Marketing Performance Dashboard** - Lead generation and source tracking
2. **Sales Pipeline Dashboard** - Pipeline monitoring and forecasting
3. **Executive Summary Dashboard** - High-level KPIs for leadership

## Folder: PayDiverse Analytics
All dashboards live in the **PayDiverse Analytics** folder in Salesforce.
