# PayDiverse Analytics Reports

## Contents
This folder contains marketing and sales analytics reports for PayDiverse 2.0.

Reports are built in Salesforce UI using specifications from:
```
PayDiverse_Reports_Spec.csv
```

## Reports:
1. **Leads_by_UTM_Source** - Monthly lead volume by marketing source
2. **Leads_by_Landing_Page** - Top converting landing pages
3. **Won_by_Acquisition_Source** - Revenue by source and campaign
4. **Pipeline_by_Acq_Campaign** - Active pipeline by campaign/stage
5. **Conversions_by_UTM_Source** - Conversion funnel by source
6. **Avg_Time_to_Close_by_Source** - Sales cycle efficiency

## Folder: PayDiverse Analytics
All reports live in the **PayDiverse Analytics** folder in Salesforce.
